2. Anschließend muss die neueste Version des Unofficial Stronghold Crusader Patch (min. 2.15a) installiert werden. Ihr findet ihn im Hauptordner. Aktuellste Version downloadbar hier: https://github.com/UnofficialCrusaderPatch/UnofficialCrusaderPatch/releases 
--> Startet den Patcher, klickt auf 'Weiter" und wählt in den Reitern "AIV", "AIC" und "Troops" meine entsprechenden Dateien aus. (Jeweils zu erkennen durch das vorangestellte "Crusader Pilaw".) Drückt dann auf "Installieren".

3. Nun können die Dateien aus den Ordnern "binks", "fx" und "gm" in deinen jeweiligen Ordner im Stronghold Crusader (Extreme) Ordner kopiert werden. Diese Ersetzung muss manuell erfolgen, ist allerdings nur einmalig notwendig.

--> Die Audiodateien müssen z. B. in den Ordner "speech", welcher sich im Ordner "fx" befindet.
--> Im gm-Ordner ist vor allem die "interface_icons2.gm1"-Datei wichtig. Diese MUSS in den gm-Ordner eures Spiels eingefügt werden. Optional habe ich euch hier einige Dateien zur Verfügung gestellt, wie ICH sie im Spiel nutze. Hier findet

By installing this mod pack you will receive my reworked vanilla-AIs, as well as my own mods (my totem AIs; Bear, Hyena, Snail and Scorpion and Inquisitor, Emira, Robber Knight and Efendi!).

Important:
Before installing the pack you should set up a copy of your current crusader installation, since there are a lot of files which will be changed/ swapped out.

Installation:

1. Download the "Herrscher des Ostens" modification.

2. After that you'll need to make sure that you have the newest version of the Unofficial Stronghold Crusader Patch (version 2.15a) installed. You will find it in the main folder. Newest version can always be downloaded here: https://github.com/UnofficialCrusaderPatch/UnofficialCrusaderPatch/releases
--> Start the patcher, click "Continue" and check in the tabs "AIV", "AIC" and "troops" the corresponding files. (Every file with "Crusader Pilaw" in the front.) Press "Install".

3. Now you can copy the data sets from the folders "binks", "fx" und "gm" in the corresponding folder in your OWN Stronghold Crusader (Extreme) game. This replacement has to be done by hand, but you'll only have to do it once.
(These files mainly replace the AIs' identity; videos, voicelines and portraits. In case you want to keep the old files, remember to save them somewhere.)
--> The audio files have to be placed in the "speech" folder, which can be found in the "fx" folder.
--> In the gm folder you'll mainly need the "interface_icons2.gm1"-file. This one HAS to be pasted into your game! Additionaly I added some other gm files in an extra folder. Here you will find skins for workshops and troops I got from the original Stronghold 1 game.
Remember to name them right by deleting the "europäisch" I added. The "gemischt"-Datei contains the workshops how I am using them: Armourer, Blacksmith and Fletcher with european skins, Poleturner and Tanner with Crusader skins.

4. Lastly you will have to copy the "cr" and "extremeTrail" files into the Stronghold Crusader main folder. (These files manage among other things all text files of the game; so it affects the names of the AIs. Again; if you want to keep the original files remember to save or rename them.)
--> In case you want to play my version of the Extreme Trail you will also have to copy all maps of the specific folder into your "extremeMaps" folder (WITHOUT renaming them!).

5. Now you can start Stronghold Crusader.
